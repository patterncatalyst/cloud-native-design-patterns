What this tutorial is

A chapter-based companion to the Designing Cloud-Native APIs deck series. Each section of the decks becomes a chapter; the code is shown for Spring Boot, Quarkus, .NET, Python, C++, and Go behind a single set of tabs, so you read the pattern once and see it in the stack you work in. The running examples are the order, payment, inventory, notification, and shipping services, on Podman locally and plain Kubernetes in production.
